# 🏨 Distance Hotel — Luxury Hotel Website

A modern, blue-themed hotel booking website built with Flask, featuring Paystack payments, admin dashboard, and responsive design.

## 🚀 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/distance-hotel.git
cd distance-hotel

# 2. Create virtual environment
python -m venv venv

# 3. Activate (Windows)
venv\Scripts\activate
# Activate (Mac/Linux)
source venv/bin/activate

# 4. Install dependencies
pip install -r requirements.txt

# 5. Set environment variables (or edit .env)
export PAYSTACK_SECRET_KEY=your_key
export PAYSTACK_PUBLIC_KEY=your_key

# 6. Run the server
python app.py
